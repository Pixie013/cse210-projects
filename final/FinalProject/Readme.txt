Library Management System
This is a library management system allows users to manage library items, patrons, and staff members.

Use the displayed menu options to perform different actions:

Add Book: Add a new book to the library.
Add DVD: Add a new DVD to the library.
Remove Item: Remove an item from the library.
Print Available Items: Display a list of available items in the library.
Borrow Item: Allow a patron to borrow an item.
Return Item: Allow a patron to return a borrowed item.

Follow the prompts to provide necessary info for each action.
